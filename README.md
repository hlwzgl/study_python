# 本仓库用于学习python基础知识

## git操作
### 初始化本地目录为git仓库
    git init
### 配置提交人信息
    git config user.name liang.hong
    git config user.email liang.hong@163.com
### 本地仓库管理到远程仓库
    git remote add origin https://github.com/hlwzgl/study_python.git
### 本地文件或文件夹加入到缓冲区
    git add {file_name or folder_name}
### 提交缓冲区文件或文件夹到本地仓库
    git commit -m "xxxxxxx"
### 推送本地仓库的文件或文件夹到远程仓库
    git pushkj l
